## Installation
### Preambula

You need to install [docker](https://docs.docker.com/engine/install/) and [ollama](https://ollama.com/download/linux)

### CLI

```bash
pip run requirements
```

## Run it

```bash
docker-compose up -d
```

```bash
ollama run mistral
```

```bash
python main.py -i <VIDEO_URL>
```
